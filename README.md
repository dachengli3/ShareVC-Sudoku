# ShareVC Sudoku
Submission for ShareVC's python challenge- Sudoku!
Built using Python 3.8 on PyCharm community

Various boards are available 'easy', 'debug', 'hard', and 'free'

Run with 
`<$ python sudoku.py --board (name)>`
